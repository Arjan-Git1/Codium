#  Codium

Codium is a text editor written in Rust, using Ratatui, Syntect, Crossterm and Piecetable crates.

# Installation

1. Download the exe.
2. Add the location of the exe in the PATH of your environment variables.
3. Use it!!!!

# Usage

1. go to cmd or any terminal. 
2. type "codium"
3. after the cursor moves to the next line, write the full name of the file you want to work with with extension.
4. happy editing!!!


# controls

1. I to go into Insert Mode
2. ESC to go into Normal Mode
3. q in normal mode to quit.
